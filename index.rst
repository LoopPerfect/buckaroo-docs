Buckaroo
====================================

Buckaroo is a dependency manager for C/C++. Using Buckaroo, it is easy to add external modules to your project in a controlled and cross-platform manner.

To use Buckaroo, you will also need to install `Buck
<https://buckbuild.com/>`_. Buck is a build system developed and maintained by Facebook, and it is used by all Buckaroo packages.
